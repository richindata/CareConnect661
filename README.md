# CareConnect661

CareConnect661 is a minimal Flutter frontend scaffold focused on left-hand
accessibility optimizations.

## Included structure

- `assets/icons/`
- `assets/images/`
- `lib/models/`
- `lib/screens/`
- `lib/state/`
- `lib/theme/`
- `lib/utils/`
- `lib/widgets/`
- `test/unit/`
- `test/widget/`

## Accessibility focus

The initial implementation keeps the most-used controls on the left edge,
supports larger touch targets, enables high-contrast styling, and exposes
adjustable text scaling for improved reach and readability.
